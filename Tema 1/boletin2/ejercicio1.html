Ejercicio 1:

Escribe un programa que solicite por pantalla un número al usuario y muestre su factorial.

  • fact(n) = n * fact(n-1)
  • fact(0) = 1
  • fact(1) = 1

Ejercicio 2:

Desarrolla un programa que le pida al usuario una cantidad de días y el programa deberá devolver cuantas horas minutos y segundos son.

Ejercicio 3:

Realiza un programa capaz de resolver una ecuación de segundo grado. Para ello, se solicitará el valor de los coeficientes a, b y c y devolverá su solución.

Ejercicio 4:

Crea un programa que calcule la letra del DNI en función del número introducido por el usuario.

Ejercicio 5:

Realiza un programa que muestre un menú al usuario con varias opciones. El usuario tendrá que introducir los valores necesarios en cada caso para obtener su solución. Esta lógica se debe repetir siempre que no se indique Salir. a. Calcular el área de un triángulo b. Calcular el área de un rectángulo c. Calcular el área del círculo d. Salir

Ejercicio 6:

Realiza un programa que muestre los números impares que no sean múltiplo de 3 ni de 7 comprendidos entre el 1 y el 100. Resuelve este programa en tres versiones distintas usando los tres bucles conocidos.
