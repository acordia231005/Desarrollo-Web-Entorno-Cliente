Boletin 1 : Typescript

Ejercicio 1

Realiza un programa que haciendo uso del objeto DATE, crees una variable para saludar al usuario con un console tras abrir la aplicación indicando la hora y la fecha actual. Haz también que se muestre por consola una hora posterior y una hora anterior.

Ejercicio 2

Crea una función que haciendo uso de las expresiones regulares permita validar el formato de una dirección de correo electrónico. Recuerda que estas son las normas que lo validan:

  1. El carácter @ es obligatorio y separa la primera y segunda parte del correo.
  2. La primera parte acepta letras minúsculas, mayúsculas, caracteres numéricos y los siguientes carácteres. #*+&’!%@?{^}”.
  3. La primera parte acepta todos los caracteres punto (.) que se deseen pero no puede ser ni el primer ni el último carácter ni tampoco pueden ir seguidos.
  4. La segunda parte acepta puntos, dígitos, guiones y letras.
